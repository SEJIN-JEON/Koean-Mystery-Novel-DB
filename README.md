#**"미스터리 장르 문학의 데이터베이스 설계를 위한 예비적 고찰: 한국 추리소설을 중심으로"**
---
이 저장소는 논문 "미스터리 장르 문학의 데이터베이스 설계를 위한 예비적 고찰: 한국 추리소설을 중심으로"에서 구축한 기초 데이터베이스를 공유하기 위해 만들어졌습니다.


📌 데이터셋 구성
본 데이터베이스는 한국 미스터리 장르의 흐름을 분석하기 위해 두 시기로 구분되어 있습니다.

P1 (태동기): 1900년대 ~ 1960년대 (오프라인 자료 디지털화-잡지 및 신문연재/단행본 중심)
P2 (확산기): 1980년대 ~ 현재 (온라인 플랫폼 스크래핑-단행본/앤솔로지 중심)

data 폴더에서 01_review는 1차 심사본에서 활용된 자료이며, 02_final은 심사 후 수정본이며 1차 심사본의 자료를 RDB로 전환하며 샘플 모델링을 진행한 자료입니다. 

📊 데이터베이스 명세(DB Spec)
한국 추리소설의 DB 명세서는 다음과 같은 데이터 구조를 가집니다.(상세 내용은 02_final의 DB spec 파일 확인)

## 1. 주요 테이블 구성
* **Author (작가)**: 작가 이름(우선어), 본명, 식별번호, 생몰년, 식별번호, 작가 이칭(Foreign key) 등 작가 상세 정보
* **Work (작품)**: 작품명, 발행일/연재시작일, 연재종료일, 출판형식, 출처, LOD URI 등 작품 상세 정보
* **Publisher (발행처)**: 발행처명, 출판 매체 형태, 출판사 이칭(FK) 등 발행처 상세 정보
* **관계 테이블(Joint Table)**: 주요 테이블 간 관계 테이블(Joint Table)을 설정하여 다대다 관계 표기
